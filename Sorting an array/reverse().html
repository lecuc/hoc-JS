JavaScript Arrays

The reverse() Method

The reverse() method reverses the elements in an array:

//Nếu bạn muốn giữ nguyên mảng gốc, hãy sao chép trước: const numbers = [1, 2, 3]; const reversed = [...numbers].reverse(); // hoặc: numbers.slice().reverse(); console.log(reversed); // [3, 2, 1] console.log(numbers); // [1, 2, 3] (gốc không bị thay đổi)
